All Amateurs except Novices:  
144.0-144.1 MHz: CW Only  
144.1-148.0 MHz: CW, Phone, Image, MCW, RTTY/Data  
Offset 600 KHz  

**70 Centimeters**  
All Amateurs except Novices:  
420.0-450.0 MHz: CW, Phone, Image, MCW, RTTY/Data
